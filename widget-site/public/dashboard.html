Your Discord connection

This is what lets your own lookups get pushed onto your own Discord profile. It's separate per person — each friend sets this up once with their own values. There are two parts: one you do by hand in your browser (Discord doesn't offer an automated way to do this part), and one this site can do for you with a single click.

Part 1 — set up your own Discord application (manual, one-time)

  1. Go to the Discord Developer Portal and click New Application.
  2. On the General Information page, copy the Application ID — paste it below.
  3. Open the Bot tab, click Reset Token, and copy the token — paste it below. Keep this private; it's encrypted before it's stored, but treat it like a password.
  4. Your Discord User ID: in Discord, enable Developer Mode (User Settings → Advanced), then right-click your own name and choose Copy User ID.
  5. Your application also needs its "Social SDK" turned on and a widget schema defined before any of this works — Discord doesn't expose this through normal settings yet, only through a browser console script run while you're logged into the Developer Portal. Follow the community widget-creator script for this part. This is the same manual step needed for the local script version — it's not something this website can safely automate, since it requires your own logged-in browser session.

Part 2 — connect your Discord account (one click, once Part 1 above is done)

Save your Application ID above first, then click this to authorize the connection on Discord's side.

Heads up: this whole feature relies on an undocumented, experimental Discord mechanism (not part of their official public API), so it can stop working without notice if Discord changes it.
